    Welcome to Droid G - Gnutella client for Android

    This android client is presented to you by Toxic-lab (HK). This client makes use of the Phex source code as the backbone to communicate with the Gnutella network. Using this android application, client can search and download files, songs, movie directly using their mobile phone. This definitely save your time compare to using Gnutella client on computer and then connect your PC and mobile phone using a USB wire and then transfer the file.

    Warning

    The Gnutella network is not working properly because the LimeWire has been torn down in 2010.1

    Development

    This project was developed initially using Eclipse back in 2009. This project has been reformed to become an IntelliJ IDEA Project. You can download a copy of IntelliJ IDEA Community Edition at https://www.jetbrains.com/idea/download/.

    This project is developed using the following Java libraries:

    • Phex project http://www.phex.org/
    • Bouncy Castle https://www.bouncycastle.org/java.html
    • Commons Codec https://commons.apache.org/proper/commons-codec/
    • Commons Collection https://commons.apache.org/proper/commons-collections/
    • Commons Http Client http://hc.apache.org/httpclient-3.x/
    • Commons Lang https://commons.apache.org/proper/commons-lang/
    • Logback http://logback.qos.ch
    • xSocket http://www.xsocket.org
